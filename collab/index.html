📣 About the Program

The Gyaansatra X Anonymous India Internship Program is a 45-day online cybersecurity internship to build practical knowledge through real-world projects, expert mentorship, and community collaboration.

✨ Program Highlights

  • 🔐 Hands-on Training & Live Projects
  • 👨‍🏫 Expert-Led Sessions & Mentorship
  • 🌐 Real-World Cybersecurity Exposure
  • 🤝 Community Engagement & Networking

🎓 Key Learning for Interns

  • 🔐 Basics of Cybersecurity & Ethical Hacking
  • 💻 Setting up Kali Linux and vulnerable labs (DVWA, Metasploitable)
  • 🕵️ Changing IP addresses using TorNet to become Anonymous
  • 🌐 Scanning networks using Nmap
  • 🛡️ Performing SQL Injection attacks
  • ⚠️ Exploiting XSS & HTML Injection vulnerabilities
  • 🔫 Using Metasploit to exploit open ports
  • 🪟 Taking over Windows systems & bypassing Defender
  • 🧰 Using tools like Villain, msfvenom, mimikatz, winpeas
  • 📈 Privilege escalation & maintaining access on compromised systems
  • 🧪 Completing real-world hacking labs on TryHackMe
  • 🎯 Pivoting attacks across isolated networks
  • 🔁 Transferring files using port forwarding
  • ⚙️ Creating automated recon tools using Bash scripting
  • 🔍 Manual bug hunting on live websites (SQLi, XSS, etc.)
  • 🚧 Bypassing IDS using fragmented ping techniques
  • ☁️ Bypassing Cloudflare Protection using encoded payload
  • 📝 Writing professional vulnerability reports
  • 🤖 Jail Breaking or hacking ChatGPT to create or enhance malicious scripts/payloads
